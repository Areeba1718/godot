159  // This is the last instance's saved state, as provided at creation time.
160  // It is NULL if there was no state. You can use this as you need; the
161  // memory will remain around until you call android_app_exec_cmd() for
162  // APP_CMD_RESUME, at which point it will be freed and savedState set to NULL.
163  // These variables should only be changed when processing a APP_CMD_SAVE_STATE,
