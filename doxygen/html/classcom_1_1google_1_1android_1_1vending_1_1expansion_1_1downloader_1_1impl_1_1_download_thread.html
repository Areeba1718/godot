HttpHost com.google.android.vending.expansion.downloader.impl.DownloadThread.getPreferredHttpHost ( Context  context,
String  url 
)
inline

Returns the preferred proxy to be used by clients. This is a wrapper around android.net.Proxy#getHost(). Currently no proxy will be returned for localhost or if the active network is Wi-Fi.

Parameters
contextthe context which will be passed to android.net.Proxy#getHost()
urlthe target URL for the request
Note
Calling this method requires permission android.permission.ACCESS_NETWORK_STATE
Returns
The preferred proxy to be used by clients, or null if there is no proxy.
