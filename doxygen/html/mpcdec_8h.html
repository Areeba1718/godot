Calls mpc_decoder_scale_output to set the scaling factor according to the replay gain stream information and the supplied ouput level

Parameters
dpointer to a musepack demuxer
levelthe desired ouput level (in db). Must be MPC_OLD_GAIN_REF (64.82 db) if you want to get the old replaygain behavior
use_gainset it to MPC_TRUE if you want to set the scaling factor according to the stream gain
use_titleMPC_TRUE : uses the title gain, MPC_FALSE : uses the album gain
clip_preventionMPC_TRUE : uses cliping prevention
