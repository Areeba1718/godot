void GodotInstance::HandleMessage ( const pp::Var &  var_message)
virtual

Called by the browser to handle the postMessage() call in Javascript. Detects which method is being called from the message contents, and calls the appropriate function. Posts the result back to the browser asynchronously.

Parameters
[in]var_messageThe message posted by the browser. The possible messages are 'fortyTwo' and 'reverseText:Hello World'. Note that the 'reverseText' form contains the string to reverse following a ':' separator.
