Detailed Description

Client library for Android Market license verifications.

The LicenseChecker is configured via a Policy which contains the logic to determine whether a user should have access to the application. For example, the Policy can define a threshold for allowable number of server or client failures before the library reports the user as not having access.

Must also provide the Base64-encoded RSA public key associated with your developer account. The public key is obtainable from the publisher site.
