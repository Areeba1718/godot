Detailed Description

This class is used by the client activity to proxy requests to the Downloader Service.

Most importantly, you must call CreateProxy during the IDownloaderClient#onServiceConnected callback in your activity in order to instantiate an IDownloaderService object that you can then use to issue commands to the DownloaderService (such as to pause and resume downloads).
